What is Node.js?

Overview

  • JavaScript runtime
  • cross-platform (V8 engine)
  • asynchronous and event driven
  • single threaded
  • open source (MIT)
Basically: Run JavaScript server sided

npm (node package manager)

  • default Node.js package manager (not required)
  • handles project dependencies and version
  • written in and for JavaScript

Important npm packages

  • express Fast, unopinionated, minimalist web framework for Node.js
  • socket.io Realtime web applications with express
  • hapi Scalable applications, with minimal overhead, out-of-the-box functionality
  • koa next generation web framework for Node.js

Create Web-Application

Dependency

> npm install express

app.js

const express = require('express')
const app = express()
const port = 3000

app.get('/', (req, res) => res.send('Hello World!'))

app.listen(port, () => console.log(`Example app listening
                                    on port ${port}!`))

express routing

app.get('/user/:userId/game/:gameId', (req, res) => {
    res.send(req.params)
})
Subset of regex is supported: 
app.post(/.*fly$/, (req, res) => {
  res.send('/.*fly$/') // matches anything that ends with fly
})

Database integration (MongoDB)

> npm i mongodb
const MongoClient = require('mongodb').MongoClient;
const url = 'mongodb://localhost/';
MongoClient.connect(url, (err, client) => {
    const db = client.db('database');
    db.collection('collection').find().each((err, doc) => {
        console.log(doc); // doc -> json object
    });
});

What's up with node.js?

Node.js in 2020

is it still worth it?
Let's narrow it down

Strengths

  • All JavaScript advantages mentioned
  • Portability of modules with package managers and V8
  • JavaScript for front- and backend
    • You're technically a full-stack developer
  • Anyone can publish a library to npm
  • Open source

Strengths part 2

  • Caching inside of node.js (save RAM)
  • Non-blocking I/O
  • Extensibility

Weaknesses

  • All JavaScript disadvantages mentioned except clients
  • Version changes may be backwards-incompatible
  • ANYONE can publish a library to npm
  • node.js is single-threaded

So why learn node.js development?

  • high demand by companies
  • large comanies use it
    • Uber
    • Netflix
    • Paypal
    • eBay
    • NASA
  • Advantages outskill the disadvantages
